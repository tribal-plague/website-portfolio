
  # Portfolio website design

  This is a code bundle for Portfolio website design. The original project is available at https://www.figma.com/design/ZWDWBziOfOcYqen8k2n6HG/Portfolio-website-design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ### Contact form setup

  The contact form submits to Supabase. Create a local `.env` file with `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`, then restart the dev server.
